Cadastro de Clientes

Este projeto é uma aplicação web desenvolvida para gerenciamento de clientes, implementada com Flask no backend e Bootstrap no frontend. A aplicação demonstra o uso completo das operações CRUD (Create, Read, Update, Delete) em um banco de dados SQLite, permitindo: Criar novos registros de clientes com informações como nome, e-mail, telefone e endereço. Visualizar a lista de clientes cadastrados em uma interface amigável. Editar dados existentes para manter as informações atualizadas. Excluir registros de clientes não mais necessários. O projeto também foi estruturado para ser responsivo e funcional em dispositivos móveis, utilizando Bootstrap para estilização. A aplicação foi hospedada online, permitindo que qualquer pessoa possa testá-la diretamente.
