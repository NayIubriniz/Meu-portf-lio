Projeto Clone Instagram

Clone do Instagram com Tailwind CSS (Tutorial no YouTube) Neste projeto, segui um tutorial no YouTube para recriar a interface de login do Instagram utilizando HTML, CSS e Tailwind CSS. A aplicação possui um layout responsivo, adaptando-se bem a diferentes tamanhos de tela. Ela inclui uma seção de login com imagens de marca e links para funcionalidades como recuperação de senha e acesso ao aplicativo. A estrutura foi construída com o uso da grid do Tailwind, garantindo uma organização eficiente dos elementos. O design é uma recriação fiel do visual do Instagram, focando na simplicidade e clareza da navegação. O uso do Tailwind CSS facilita o desenvolvimento e personalização dos estilos de forma rápida e eficiente.
